# bcrypt-password-hash-verifier

A Laravel application for verifying bcrypt password hashes.

## Requirements

- PHP >= 8.2
- Laravel 12
- Node.js 22

## Installation

1. Clone the repository: `git clone https://github.com/ashikurweb/bcrypt-password-hash-verifier.git`
2. Copy the environment file: `cp .env.example .env`
3. Install dependencies: `composer install` and `npm install`
4. Generate application key: `php artisan key:generate`
5. Run the development server: `npm run dev`
6. Or build for production: `npm run build`
